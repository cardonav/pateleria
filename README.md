# Sistema de Gestión de Pedidos - Pastelería

Aplicación web para gestionar pedidos de una pastelería con tres tipos de pasteles.

## Características

- Gestión completa de pedidos (Crear, Leer, Actualizar, Eliminar)
- Tres tipos de pasteles: Básico, Mediano y Grande
- Dos estados de pedido: Recepcionado y Despachado
- Cálculo automático de totales
- Estadísticas en tiempo real
- Interfaz responsive y amigable

## Requisitos

- Servidor web (Apache, Nginx, etc.)
- PHP 7.4 o superior
- MySQL 5.7 o superior
